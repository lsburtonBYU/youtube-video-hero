HTML setup

In place of your picture element add two nested div elements: .video-background and .video-foreground. You can keep your text overlay. element. 

<div class="hero">

    <div class="video-background">
        <div class="video-foreground">
            <!-- PASTE YOUR YOUTUBE EMBED CODE HERE -->
        </div>
    </div>

    <div class="text-wrapper">
        <h1>Cleveland</h1>
        <p>No longer the mistake on the lake</p>
    </div>
</div>

Find the video you would like to use on YouTube (be aware that some videos are region-restricted).

  • Click the Share button below the video
  • Choose Embed
  • Optionally select a start time; this can fast forward over opening credits
  • Uncheck "Show player controls"
  • Click copy
  • Paste the embed code inside the .video-foreground div

We will need to remove and add some information to the embed code. A typical embed code (with no controls selected, and start at 7s will look something like this:

<iframe width="560" height="315"
    src="https://www.youtube.com/embed/9s1UpG1qo9M?controls=0&start=7"
    title="YouTube video player" frameborder="0"
    allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share"
    allowfullscreen>
</iframe>

Remove the following:

  • width and height attributes. We will set them in the CSS
  • title attribute - not needed and it would appear as a tool tip
  • frameborder is deprecated and will give a validation error
  • most of the allow attribute properties are not needed. Remove all but autoplay
  • allowfullscreen

Your embed code should now look like this

<iframe
    src="https://www.youtube.com/embed/9s1UpG1qo9M?controls=0&start=7"
    allow="autoplay">
</iframe>

Next, we want to enable autoplay (which is starting to be disabled by default on mobile phones) and mute the video. YouTube also has a "modestbranding" feature that is supposed to remove most branding. We will add all of these as parameters on the video URL. To add parameters, add a ? after the URL, then add the parameters separated by &. Here's the URL with the parameters

<iframe
    src="https://www.youtube.com/embed/9s1UpG1qo9M?controls=0&start=7&autoplay=1&mute=1&modestbranding=1"
    allow="autoplay">
</iframe>

If you want the video to loop, you need a playlist. To have just the one video loop, use the video ID as the playlist value (the video ID is right after embed/ in the URL). Add the playlist and loop=1 to the URL. Also add look to the allow attribute 

<iframe
    src="https://www.youtube.com/embed/9s1UpG1qo9M?controls=0&start=7&autoplay=1&mute=1&modestbranding=1&playlist=9s1UpG1qo9M&loop=1"
    allow="autoplay; loop">
</iframe>

CSS styling

Next, we need CSS to style the video. First, hide any overflow on your hero element. You can give it a height, but if you are using the reset include in the assignment files, remove the declaration that sets the html and body to height: 100%. 

.hero {
    position: relative;
    overflow: hidden;
    height: 90vh;
}

Here's all the CSS needed for the two .video divs:

.video-background {
    background: #000;
    position: absolute;
    inset: 0;
    z-index: -99;
  }

  .video-foreground,
  .video-background iframe {
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    pointer-events: none;
  }

  @media (min-aspect-ratio: 16/9) {
    .video-foreground {
      height: 300%;
      top: -100%;
    }
  }
  @media (max-aspect-ratio: 16/9) {
    .video-foreground {
      width: 300%;
      left: -100%;
    }
  }

The double divs act emulate background-fit: cover and the media queries size the video based on aspect ratio.
